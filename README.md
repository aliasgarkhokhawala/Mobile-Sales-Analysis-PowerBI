# Mobile Sales Analysis Dashboard

## Project Overview

This project focuses on analyzing mobile sales data using Power BI to uncover valuable business insights and support data-driven decision-making. The dashboard provides a comprehensive view of sales performance across different mobile brands, products, and regions.

## Objectives

* Analyze overall mobile sales performance.
* Identify top-performing mobile brands and models.
* Monitor revenue, quantity sold, and sales trends.
* Compare sales performance across different regions and categories.
* Create interactive visualizations for business reporting.

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Microsoft Excel

## Key Features

* Interactive dashboard with dynamic filters and slicers.
* Brand-wise and model-wise sales analysis.
* Revenue and quantity sold tracking.
* Sales trend analysis using charts and KPIs.
* Regional performance comparison.
* User-friendly visualizations for quick decision-making.

## Key Insights

* Identified top-performing mobile brands contributing the highest revenue.
* Analyzed sales patterns and trends across different periods.
* Compared regional sales performance to highlight high-performing markets.
* Evaluated product-wise sales distribution and customer preferences.
* Generated actionable insights to support business growth strategies.

## Outcome

The dashboard enables users to monitor sales performance efficiently, identify business opportunities, and make informed decisions through data visualization and analytics.
